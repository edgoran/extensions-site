Privacy Policy

Last updated: January 2025

Extension: Link Grab

Summary: Link Grab does not collect, store, or transmit any personal data. All data remains locally on your device.

Data Collection

This extension does not collect any personal information. Specifically:

  • No personal data is collected
  • No browsing history is tracked
  • No data is sent to external servers
  • No analytics or tracking tools are used
  • No cookies are set by the extension

Data Storage

Any links you grab or settings you configure within the extension are stored locally on your device using Chrome's built-in storage APIs. This data is never transmitted to any third party.

Page Content Access

Link Grab accesses the content of web pages solely to extract links as requested by you. This content is processed locally in your browser and is never sent to any external server or stored beyond your current session unless you explicitly save it.

Permissions

The extension requests only the minimum permissions necessary for its core functionality — specifically the ability to read page content to extract links. No permissions are used to collect or transmit user data.

Third-Party Services

This extension does not integrate with or send data to any third-party services, APIs, or analytics platforms.

Changes to This Policy

If this privacy policy is updated, the changes will be reflected on this page with an updated date. Significant changes will be communicated through extension updates.
