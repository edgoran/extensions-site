Privacy Policy

PatchPoint - Game Patch Notes Tracker

Last updated: 28th June 2025

Summary: PatchPoint does not collect, transmit, or store any personal data on external servers. All user data is stored locally on your device.

Data Collection

PatchPoint does NOT collect any personal information. No names, emails, browsing history, analytics, cookies, or tracking identifiers are gathered.

Local Storage

All user data is stored locally on your device using Chrome's built-in storage API:

  • Game subscriptions (which games you follow)
  • Cached patch note content
  • Custom groups and organisation preferences
  • Theme and display settings
  • Game request rate-limiting timestamps

This data never leaves your device unless you explicitly use the game request feature.

External Services

Service Purpose Data Sent
Steam Web API Fetch patch notes for Steam games Game App IDs (public, non-personal)
RAWG.io API Search games and retrieve metadata/images Search query text only
Web3Forms Submit game requests (optional user action) Game name, platform, notes (no personal info)
RSS Feeds Fetch patch notes for non-Steam games None (read-only fetch)

No personal or identifying information is sent to any of these services.

Permissions

Permission Purpose
storage Save your subscriptions, groups, and settings locally
alarms Schedule background checks for new patch notes every 30 minutes
notifications Send desktop notifications when new patches are detected
host_permissions Connect to Steam API, RAWG API, Web3Forms, and RSS feeds to fetch game data

Data Retention

All data is stored locally and persists until you:

  • Unsubscribe from a game (removes that game's cached patch notes)
  • Uninstall the extension (removes all data)
  • Clear browser extension data manually

Third-Party Sharing

We do not share, sell, or transmit any user data to third parties.

Children's Privacy

PatchPoint does not knowingly collect information from children under 13. The extension does not collect personal information from any user.

Changes to This Policy

Any changes to this privacy policy will be reflected on this page with an updated date.
